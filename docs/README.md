# Docs

This directory contains StellarTap MVP documentation focused on **primitives first** and a **regulation-aware architecture**.

- `overview.md`: What we’re building (plain language).
- `e2e-flow.md`: End-to-end user flow (payer + payee/merchant).
- `architecture.md`: System components and responsibilities.
- `protocol-request-envelope.md`: Request envelope fields + encoding.
- `sep7.md`: SEP-7-compatible payment request representation.
- `security-model.md`: Threat model and mitigations for the MVP.
- `roadmap.md`: What’s MVP vs later (push, offline, NFC depth).
