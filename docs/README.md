# Docs

This directory contains StellarTap documentation focused on **primitives first**, a **regulation-aware architecture**, and the **WebRTC tap-to-pay** product direction.

- `overview.md`: What we’re building (plain language).
- `webrtc-tap-to-pay.md`: **Canonical spec** for signaling + WebRTC + RTT proximity + implementation phases.
- `e2e-flow.md`: End-to-end user flow (payer + payee/merchant).
- `architecture.md`: System components (wallet, receiver, signaling, Horizon).
- `protocol-request-envelope.md`: Request envelope fields + encoding.
- `sep7.md`: SEP-7-compatible payment request representation.
- `security-model.md`: Threat model and mitigations.
- `roadmap.md`: Done vs next vs later.
